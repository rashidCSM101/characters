# Run files
To run a file, use the following command:
Take images from the folder `input_images` and run the following command in the terminal:
```
python batch_urdu_ocr_grouped.py
```
### output
The output will be saved in the `output_grouped` folder. 
### Note subfolders
binary_images, \
extracted_characters,\
preprocessed_steps\
visualization_images
